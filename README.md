# ローカルMDカンバン

ローカルのマークダウンファイルを使ったカンバン形式のタスク管理アプリ。

## 特徴

- マークダウンファイルでタスクを管理（データはローカルに保存）
- ドラッグ＆ドロップでステータス変更
- ステータス別の色分け（未着手:赤、着手中:黄、完了:緑）
- サブタスク、優先度、期限、担当者の管理
- ソート機能（作成日時、更新日時、期限、優先度）

## 必要環境

- Node.js 18+
- pnpm
- Rust

## 開発

```bash
# 依存パッケージをインストール
pnpm install

# 開発サーバーを起動
pnpm tauri dev
```

## ビルド

```bash
# プロダクションビルド
pnpm tauri build
```

ビルド成果物は `src-tauri/target/release/bundle/` に出力されます。

## 使い方

1. アプリを起動
2. 「既存フォルダを開く」または「新規フォルダを作成」を選択
3. カンバンボードでタスクを管理
   - 「+ 新規タスク」でタスク作成
   - タスクをドラッグしてステータス変更
   - タスクをクリックして編集

## マークダウン形式

タスクは以下の形式で保存されます：

```markdown
# タスク名

## メタデータ

- created: 2026-02-03-10:00
- updated: 2026-02-03-10:00
- status: 未着手
- priority: 低
- due: 2026-02-09
- assignee: -

## サブタスク

- [ ] サブタスク1
- [x] サブタスク2

## メモ

自由に記述
```
